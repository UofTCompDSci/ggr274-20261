Working With Survey Data#

Featuring: Your Survey Responses!#

Surveys#

Social science data is often obtained through surveys. Surveys are valuable because they provide us with a way to collect actual information about people. This applies across many different social science domains!

  • Economics: What is your current income?

  • Political Science: Which political party do you currently plan to vote for?

  • Linguistics: What languages do you speak?

  • Psychology: What are your preferences?

  • Sociology: What is your religious affiliation?

  • Geography: Where do you live, where do you work, and how do you travel between the two?

Surveys#

Surveys are usually drawn from a sample of the population.

  • Why? Because it is very difficult to survey every single person!

  • An exception is the Census, but they have to work really hard to get everybody to respond.

  • We will talk later about how to handle the challenge of using a sample instead of an entire population. For now, let‚Äôs focus on the data from our in-class survey!

Class survey!#

We sent out a survey last week, asking you to provide answers to some basic questions. Let‚Äôs see what you all said!

These survey responses are stored in a file called ggr274_survey.csv. This file contains a table of survey responses, consisting of:

  • A row for each person who responded

  • A column for each question that we asked

Reading our data: open and readlines#

Now that we‚Äôve seen the file, let‚Äôs read the file‚Äôs contents into Python.

Once again, we can do this in two steps:

  1. Open the file.

  2. Read the file data into Python, line by line.

survey_file = open("ggr274_survey.csv", encoding="utf-8-sig")

survey_data = survey_file.readlines()

Exploring Our Data#

Let‚Äôs look at just the first line from the file:

survey_data[0]

We need to split these values into individual pieces. We can do this using .split(",")

print(survey_data[0].split(","))

This first row appears to be the header: this contains the names for each of our columns! For now, let‚Äôs just check out the first actual row of our data:

print(survey_data[1].split(","))

Summarizing numeric data#

Now we are ready to pull out the data from a specific column! Let‚Äôs use the same strategy we used to pull the population column from the electoral data, except this time for the commute_time column in survey_data:

populations = []

for line in district_data:
    entries = line.split(",")
    
    population_entry = entries[1].strip()
    
    population_int = int(population_entry)
    
    populations.append(population_int)

In addition to changing the variable names, we need to make a few other changes:

  1. Instead of for line in district_data, we are going to start our for loop with for line in survey_data[1:]. Why?

  1. Instead of population_entry = entries[1].strip(), we are going to use commute_entry = entries[9].strip(). Why?

Let‚Äôs look at the commute times!

commutes = []

for line in survey_data[1:]:
    entries = line.split(",")
    
    commute_entry = entries[9].strip()
    
    commute_int = int(commute_entry)
    
    commutes.append(commute_int)
    
print(commutes)

Now that we have a column from our data, let‚Äôs look at how many responses we have using len().

num_responses = len(commutes)

print(f"Number of responses: {num_responses}.")

We can also calculate the ‚Äúresponse rate‚Äù, by dividing the number of responses by the total number of students in the class (120):

print(f"Response rate: {num_responses/120}")

We can also compute summary statistics for the list of commute times!

sum_commute = sum(commutes)
max_commute = max(commutes)
min_commute = min(commutes)
avg_commute = sum_commute / num_responses

print(f"Sum of all commute times: {sum_commute}.")
print(f"Minimum commute: {min_commute}.")
print(f"Maximum commute: {max_commute}.")

Summarizing categorical data#

Now let‚Äôs take a look at a categorical variable. How many people said that Winter was their favourite season?

This time, we‚Äôll create a new list called seasons using the index [5] (for ‚Äúfavourite_season‚Äù, the sixth column in our table):

seasons = []

for line in survey_data[1:]:
    entries = line.split(",")
    
    season_entry = entries[5].strip()
        
    seasons.append(season_entry)
    
print(seasons)

How do we figure out how many people said their favourite season was winter? We can use a for loop to help us determine this.

Let‚Äôs start by setting the count of the number of people that prefer winter (winter_count) at 0.

Next, we‚Äôll add +1 for each time that ‚ÄúWinter‚Äù appears in seasons:

winter_count = 0

for season in seasons:
    if season == "Winter":
        winter_count = winter_count + 1

Now we can calculate how many people responded ‚ÄúWinter‚Äù, as well as the share of total responses!

print(f"Number of responses for Winter: {winter_count}.")
print(f"Share of responses for Winter: {winter_count/len(seasons)}.")

Using dictionaries to summarize groups#

What if we want to compare the responses of different groups? For example, we asked everyone what their favourite hot drink was (coffee vs. tea), and how many cups they had per week.

Did coffee drinkers have more cups per week, or tea drinkers?

To answer this question, we‚Äôre going to need a dictionary. This dictionary will contain separate lists for each type of drink (coffee, tea, etc.).

# Create an empty dictionary
drink_dict = {}

We‚Äôre going to use a for loop to create multiple categories inside of drink_dict. For each row in the data:

  1. We look at the value in the drink_type column (index [3])

  2. If that drink type is not in our dictionary, we create a new key in our dictionary for that drink type.

  3. We look up the number of cups_per_week for that row (index [4]) and convert to an integer (cups_int)

  4. We append the value of cups_int to the correct category of our dictionary (drink_dict[drink_type])

# Read each line of survey_data (skipping line 0)
for line in survey_data[1:]:
    entries = line.split(",")
    
    # Column of drink types
    drink_type = entries[3].strip()

    # Check if this drink_type is already in our dictionary
    # If not, add a new key in the dictionary for this drink_type!
    if drink_type not in drink_dict:
        drink_dict[drink_type] = []

    # Column of cups per week (convert to integer)
    cups_entry = entries[4].strip()
    cups_int = int(cups_entry)

    # Add number of cups to the correct drink type for this row
    drink_dict[drink_type].append(cups_int) 
        
print(drink_dict)

We can now pull out an individual group containing only the responses from tea drinkers or coffee drinkers!

print(drink_dict["Tea"])

print(drink_dict["Coffee"])

Now, let‚Äôs see whether tea or coffee drinkers have more cups per week!#

total_tea_cups = sum(drink_dict["Tea"])
total_tea_drinkers = len(drink_dict["Tea"])
average_tea_cups = total_tea_cups/total_tea_drinkers


print(f"Total tea cups per week: {total_tea_cups}")
print(f"Total tea drinkers: {total_tea_drinkers}")
print(f"Average cups for tea drinkers: {average_tea_cups}")

total_coffee_cups = sum(drink_dict["Coffee"])
total_coffee_drinkers = len(drink_dict["Coffee"])
average_coffee_cups = total_coffee_cups/total_coffee_drinkers

print(f"Total coffee cups per week: {total_coffee_cups}")
print(f"Total coffee drinkers: {total_coffee_drinkers}")
print(f"Average cups for coffee drinkers: {average_coffee_cups}")
